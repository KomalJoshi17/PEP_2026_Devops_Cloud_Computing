ls

initialise vagrant with ubuntu box
vagrant init bento/ubuntu-22.04
VagrantFile is created

configuration of VagrantFile
# -*- mode: ruby -*-
# vi: set ft=ruby :

Vagrant.configure("2") do |config|

  config.vm.box = "bento/ubuntu-22.04"

  config.vm.provider "virtualbox" do |vb|
    vb.name   = "ubuntu-devops"
    vb.memory = "2048"
    vb.cpus   = 2
    vb.gui    = false
  end

end

Vagrant.configure("2") do |config|
  config.vm.box = "bento/ubuntu-22.04"

  config.vm.provider "virtualbox" do |vb|
    vb.memory = "1024"
    vb.cpus = 1
  end
end

Vagrantfile → Manages all VM configurations.
vagrant up → Creates and starts the VM.
vagrant ssh → Connects to the VM.
vagrant halt → Stops the VM.
vagrant destroy → Deletes the VM.
